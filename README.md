# TCPyPot

A TCP honeypot written in python

## Setup

Install python3

## Running

### Starting the program

Linux

```bash
python3 main.py
```

Windows

```bash
python main.py
```

### Port forwarding

Forward port 5050 of the machine you are running this script in

### Wait for logs

When someone connects, it will be logged in the console and in the log.txt file
